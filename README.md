Personal Trainer Landing Page

A modern, responsive landing page for a **personal trainer or fitness coach**, designed using **HTML5**, **CSS3**, and **Bootstrap 5**.  
This project is built to showcase personal training services, contact information, and professional branding in a clean and client-friendly layout.

Features
Fully responsive design for all screen sizes  
Fixed sticky navbar for easy navigation  
Hero section with background image and motivational headline  
Service section displaying multiple fitness programs  
Contact form with professional styling  
Footer with social media icons  
Smooth hover transitions and animations  
Mobile-friendly and fast-loading
